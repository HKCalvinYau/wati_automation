變數功能使用指南

什麼是變數？

變數是模板中的動態內容，使用 {{變數名稱}} 格式。

親愛的 {{name}}， 您的寵物 {{pet_name}} 的預約時間是 {{date}} {{time}}。

如何使用變數功能

1. 添加變數

在模板內容中使用 {{變數名稱}} 格式，系統會自動識別

2. 輸入變數值

在「變數值輸入」區塊找到對應變數，在輸入框中輸入實際值

3. 即時預覽

預覽區塊會即時顯示已帶入變數值的完整訊息

4. 複製內容

點擊「複製預覽內容」按鈕，複製已帶入值的完整訊息

常見變數說明

{{name}} - 收件人姓名 {{date}} - 日期 {{time}} - 時間 {{phone}} - 電話號碼 {{email}} - 電子郵件 {{pet_name}} - 寵物名稱 {{service}} - 服務項目

常見問題

如何添加新的變數？

在模板內容中直接輸入 {{變數名稱}}，系統會自動識別。

如何重置變數值？

點擊「重設」按鈕可以清空所有變數值。

複製預覽內容會統計使用次數嗎？

是的，複製預覽內容會自動增加使用次數。

如何切換語言？

使用語言切換按鈕，或使用快捷鍵 Alt+Z（中文）和 Alt+E（英文）。

快捷鍵

Alt + Z - 切換到繁體中文 Alt + E - 切換到英文 Ctrl + F - 搜索模板 Ctrl + C - 複製內容（在預覽區塊）
