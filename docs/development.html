開發指南

寵物善終服務 WhatsApp 訊息模板管理系統的完整開發文檔

系統概覽

寵物善終服務 WhatsApp 訊息模板管理系統是一個現代化的 Web 應用程式，專為寵物善終服務業者設計，提供完整的訊息模板管理解決方案。

主要功能

  • 雙語支援（繁體中文/英文）
  • 智能搜索和過濾
  • 響應式設計
  • 使用次數統計
  • 圖片管理
  • 批量操作

技術架構

技術棧

  • 前端：HTML5, CSS3, JavaScript (ES6+)
  • 後端：PHP 8.0+
  • 容器化：Docker & Docker Compose
  • Web 服務器：Apache

目錄結構

wati_automation/
├── index.html                    # 首頁

API 文檔

基礎 URL

所有 API 端點的基礎 URL：http://localhost:9090/api/

API 端點

  • GET /get-templates.php - 獲取所有模板列表
  • POST /save-template.php - 保存或更新模板
  • POST /save-template-images.php - 保存模板相關圖片
  • POST /increment-usage.php - 增加模板使用次數
